### 1. Core & Form Elements

Essential components for user input and basic actions.

```bash
npx shadcn-vue@latest add button button-group badge input input-group label textarea checkbox radio-group switch select native-select

```

### 2. Layout & Structure

Components used to organize your page layout and containers.

```bash
npx shadcn-vue@latest add card separator scroll-area aspect-ratio resizable skeleton sidebar sheet drawer

```

### 3. Navigation

Components for moving between pages or sections.

```bash
npx shadcn-vue@latest add breadcrumb navigation-menu pagination tabs stepper menubar

```

### 4. Overlays & Feedback

Modals, tooltips, and notification components.

```bash
npx shadcn-vue@latest add dialog alert-dialog popover tooltip context-menu dropdown-menu hover-card sonner alert

```

### 5. Advanced Inputs

Complex input types for dates, numbers, and specialized data.

```bash
npx shadcn-vue@latest add calendar range-calendar number-field input-otp tags-input slider command

```

### 6. Data Display & Indicators

Components for visualizing data and showing status.

```bash
npx shadcn-vue@latest add table chart accordion collapsible progress spinner avatar kbd empty

```
